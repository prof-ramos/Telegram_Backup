Visão Geral

O Telegram Backup CLI é um sistema profissional desenvolvido em Python que utiliza a biblioteca Telethon para automatizar o backup de mensagens entre chats do Telegram. O sistema oferece backup em tempo real, filtros inteligentes e uma interface visual elegante usando a biblioteca Rich.

Telethon

Biblioteca Python para automação do Telegram

Rich CLI

Interface visual com tabelas e cores

Click

Framework para criação de comandos CLI

Arquitetura do Sistema

Componentes Principais

1

TelegramClient

Conexão principal com a API do Telegram usando Telethon

2

Event Handler

Processa eventos de novas mensagens em tempo real

3

Filter Engine

Aplica filtros de mídia e conteúdo configurados

4

State Manager

Gerencia estado persistente para evitar duplicações

Fluxo de Execução

# Fluxo principal do sistema
1. Carregar configuração (config.json, config.env)
2. Conectar ao Telegram (TelegramClient)
3. Resolver entidades de chat
4. Configurar event handlers
5. Executar backup histórico
6. Iniciar monitoramento em tempo real
7. Manter estado persistente

Configuração

config.env

Credenciais da API do Telegram

# Credenciais da API do Telegram
API_ID=sua_api_id
API_HASH=sua_api_hash
SESSION_NAME=backup_premium

config.json

Rotas e filtros de backup

{
"routes": {
"123456789": "me",
"@meu_canal": "backup_chat_id"
},
"filters": {
"media_only": false,
"photos": true,
"videos": true
}
}

Opções de Filtros

media_only

Apenas mensagens com mídia (fotos, vídeos, documentos)

photos

Incluir fotos no backup

videos

Incluir vídeos no backup

text

Incluir mensagens de texto (quando media_only=false)

API Reference

Funções Principais

load_config()

Carrega configuração de rotas e filtros do config.json

# Retorna: tuple(routes: dict, filters: dict)
routes, filters = load_config()

resolve_route_names(routes)

Resolve nomes amigáveis para chats usando Telethon

# Retorna: dict com nomes formatados
names = await resolve_route_names(routes)

should_backup(message)

Verifica se mensagem deve ser backupada baseado nos filtros

# Retorna: bool
if should_backup(message):
await forward_message()

backup_history_for_chat(src_entity, dst_entity)

Faz backup histórico de um chat específico

# Processa mensagens antigas
await backup_history_for_chat(source, destination)

Exemplos de Uso

Exemplo 1: Backup Básico

# Configurar rota simples
python backup_cli.py add-route
# Selecionar origem: @meu_canal
# Selecionar destino: me
# Executar backup
python backup_cli.py run

Exemplo 2: Backup com Filtros

# Configurar apenas mídia
python backup_cli.py menu
# Opção 3: Configurar filtros
# media_only: true
# photos: true
# videos: true

Exemplo 3: Múltiplas Rotas

# Adicionar múltiplas rotas
python backup_cli.py add-route
# Origem: @canal_tech → Destino: backup_tech
python backup_cli.py add-route
# Origem: @canal_news → Destino: backup_news
# Verificar configuração
python backup_cli.py show-config

Solução de Problemas

Problemas Comuns

Erro de conexão com Telegram

ConnectionError: Unable to connect

Solução:

  • • Verifique sua conexão com a internet
  • • Confirme as credenciais API_ID e API_HASH
  • • Verifique se não há bloqueios de firewall

Sessão expirada

SessionRevokedError: The authorization has been revoked

Solução:

  • • Delete o arquivo de sessão (.session)
  • • Execute o script novamente para nova autenticação
  • • Verifique o código de confirmação no Telegram

Erro ao encaminhar mensagens

ChatWriteForbiddenError: You can't write in this chat

Solução:

  • • Verifique permissões no chat de destino
  • • Confirme se o bot pode enviar mensagens
  • • Verifique se não há restrições no chat

Dicas de Performance

  • Use filtros para reduzir o volume de mensagens processadas
  • Configure poucas rotas simultâneas para melhor performance
  • Monitore o uso de memória com muitos chats
  • Use sessões dedicadas para diferentes propósitos
