Funcionalidades Avançadas

Sistema completo com automação inteligente, filtros personalizáveis e interface profissional

Backup em Tempo Real

Monitoramento contínuo de chats com encaminhamento automático de novas mensagens usando eventos do Telethon.

Filtros Inteligentes

Filtros personalizáveis para mídia, fotos, vídeos e mensagens de texto com configuração flexível via CLI.

Interface Rich CLI

Interface visual elegante com tabelas, cores e menus interativos usando a biblioteca Rich para Python.

Gestão de Rotas

Sistema completo para adicionar, remover e configurar rotas de backup entre chats com nomes amigáveis.

Estado Persistente

Controle de estado com JSON para evitar duplicações e manter histórico de mensagens processadas.

Configuração Flexível

Arquivo JSON para configuração de rotas e filtros com suporte a múltiplos formatos de identificação.

Instalação Rápida

Configure o sistema em minutos com nosso guia passo a passo

Dependências

# Instalar dependências
pip install telethon python-dotenv click rich

Configuração

# Criar config.env
API_ID=sua_api_id
API_HASH=sua_api_hash
SESSION_NAME=backup_premium

Primeiros Passos

1

Configurar API do Telegram

Obtenha suas credenciais em my.telegram.org e configure no arquivo config.env

2

Executar menu interativo

Use python backup_cli.py menu para acessar a interface completa

3

Configurar rotas de backup

Adicione rotas com python backup_cli.py add-route

4

Iniciar backup

Execute o backup em tempo real com python backup_cli.py run

Documentação Completa

Explore todos os comandos e funcionalidades do sistema

Comandos Principais

python backup_cli.py menu

Interface interativa completa com menu Rich

python backup_cli.py run

Executa backup direto com configurações atuais

python backup_cli.py show-config

Mostra configuração atual com nomes dos chats

python backup_cli.py add-route

Adiciona nova rota de backup interativamente

python backup_cli.py remove-route

Remove rotas de backup configuradas

Estrutura de Arquivos

config.env

Credenciais da API do Telegram (API_ID, API_HASH, SESSION_NAME)

config.json

Configuração de rotas e filtros de backup

backup_state.json

Estado do backup para evitar duplicações

backup_cli.py

Script principal do sistema de backup
